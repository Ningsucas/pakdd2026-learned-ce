# PAKDD 2026 Tutorial: Learning-based Cardinality Estimation

This repository provides tutorial materials for the PAKDD 2026 tutorial on learning-based cardinality estimation across relational, graph, and multi-model data.

The tutorial uses representative methods to demonstrate how a shared analytical framework localizes design choices across different data models and workloads.
